# 📊 Unemployment Analysis with Python

## Overview

This project is completed as **Task 2** of the **Data Science Internship** at **Oasis Infobyte**.

The objective of this project is to analyze unemployment trends in India using Python and perform Exploratory Data Analysis (EDA) to understand regional and temporal unemployment patterns, including the impact of the COVID-19 pandemic.

---

## Dataset

- **Dataset:** Unemployment in India
- **Source:** Kaggle
- **Records:** Indian unemployment statistics
- **Features:**
  - Region
  - Date
  - Frequency
  - Estimated Unemployment Rate (%)
  - Estimated Employed
  - Estimated Labour Participation Rate (%)
  - Area

---

## Objectives

- Load and clean the dataset
- Explore unemployment trends
- Analyze unemployment across different regions
- Study monthly unemployment patterns
- Compare unemployment before and after COVID-19
- Visualize important insights using charts
