# 🚗 Car Price Prediction with Machine Learning

## 📌 Overview

This project is completed as **Task 3** of the **Data Science Internship** at **Oasis Infobyte**.

The objective of this project is to build a Machine Learning regression model that predicts the selling price of a used car based on various features such as brand, manufacturing year, fuel type, transmission, mileage, engine capacity, and other vehicle characteristics.

---

## 📂 Dataset

- **Dataset:** Vehicle Dataset from CarDekho
- **Source:** Kaggle
- **Records:** Used car listings
- **Features:**
  - Car Name
  - Manufacturing Year
  - Selling Price
  - Kilometers Driven
  - Fuel Type
  - Seller Type
  - Transmission
  - Owner Type
  - Mileage
  - Engine
  - Max Power
  - Torque
  - Seats

---

## 🎯 Objectives

- Load and explore the dataset
- Clean and preprocess the data
- Handle missing values and duplicates
- Perform feature engineering
- Analyze relationships between different features

## 🤖 Machine Learning Models

The following regression models were trained and evaluated:

- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

### Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

The best-performing model was selected based on these evaluation metrics.

---

## 🚀 Results

The project successfully predicts the selling price of used cars using machine learning techniques.

Key observations include:

- Car age significantly affects selling price.
- Premium brands generally have higher resale values.
- Fuel type and transmission influence vehicle prices.
- Random Forest Regressor achieved better prediction performance than Linear Regression.
