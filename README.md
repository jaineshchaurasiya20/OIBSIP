# 🚀 Completed Projects

## 📌 Task 1 — Iris Flower Classification

### Objective

Build a machine learning classification model that predicts the species of an Iris flower based on its physical measurements.

### Skills Learned

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Selection
- Classification Models
- Model Evaluation

### Technologies

- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📌 Task 2 — Unemployment Analysis with Python

### Objective

Perform exploratory data analysis on India's unemployment dataset to identify regional and temporal trends, including the impact of the COVID-19 pandemic.

### Skills Learned

- Data Cleaning
- Missing Value Analysis
- Exploratory Data Analysis
- Time Series Analysis
- Correlation Analysis
- Data Visualization
- Statistical Interpretation

### Technologies

- Pandas
- Matplotlib
- Seaborn

---

## 📌 Task 3 — Car Price Prediction with Machine Learning

### Objective

Develop regression models to predict the selling price of used cars based on various vehicle features.

### Skills Learned

- Feature Engineering
- Data Preprocessing
- Regression Algorithms
- Feature Encoding
- Model Evaluation
- Feature Importance Analysis

### Machine Learning Models

- Linear Regression
- Random Forest Regressor

### Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
